## Python Basics
### Variables
Variables store data values.

Example:
name = "Taylor"
### Control Flow
Use if/else statements to branch logic.

Example:
if score >= 80:
    print("Great job!")
### Functions
Functions help you reuse code.

Example:

def greet(user):
    return f"Hello, {user}!"
